# Loan & CrowdFunding - Future of Finance

Loan & CrowdFunding (UniFund) connects brilliant students with alumni investors. It is Bangladesh's first merit-based funding platform where students can secure funding based on their CGPA and potential, without the need for family collateral.

## 🚀 Features

### 🎓 For Students
- **Profile Creation**: Sign up and verify student status with university ID.
- **Trust Score**: AI-driven analysis of academic performance to assign a credit/trust score.
- **Funding Options**:
  - **Loans**: Interest-free (Qard Hasan) or profit-sharing based loans.
  - **Crowdfunding**: Launch campaigns for tuition or projects.
- **Repayment Tracking**: Transparent dashboard to track installments and build credit history.

### 🤝 For Donors & Alumni
- **Marketplace**: Browse student campaigns and filter by university or cause.
- **Direct Impact**: Donate directly to students or projects you care about.
- **Transparency**: Tracking tools to see how funds are utilized and repaid.
- **Community**: Network with other successful alumni.

### 🛡️ Admin & Security
- **Admin Dashboard**: Overview of total funds, active users, and system health.
- **Fraud Center**: Automated alerts for suspicious activities (e.g., multiple IPs, rapid applications).
- **User Management**: Tools to verify, block, or assist users.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS
- **Backend**: PHP 8.0+
- **Database**: MySQL 5.7+
- **Styling**: Tailwind CSS (via CDN) + Custom CSS
- **Icons**: Lucide Icons

## 📂 Project Structure

- **`api/`**: Contains PHP scripts for handling backend requests (login, registration, data fetching).
- **`includes/`**: Shared PHP files like database connection (`db.php`).
- **`*.html`**: The frontend views for different dashboards and pages.
- **`*.js`**: JavaScript files containing the logic for each page (API calls, DOM manipulation).
- **`setup.sql`**: The full database schema and initial seed data.
